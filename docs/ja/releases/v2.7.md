# リリース v2.7

## 変更点

---

### センサ処理

- コンパスキャリブレーションの手法を最大最小から楕円体近似に

### モデリング

- エンジンのダイナミクスの推定機能を実装
- 可変ピッチプロペラの空力特性の推定機能を実装

### 制御

- Y軸まわりにのみチルトするマルチコプターに対応
- 非平面ロータ配置マルチコプターのミキサーを改善
- マルチコプターのゲインを二次系の標準形だけでなく2段階のカスケードでも設定可能に

### 通信

- GCSからFCへの接続をログインパスワードからSSH鍵認証に変更
- シミュレーションの終了直後に再度立ち上げるとROSの通信ができないバグを解決

### その他

- 各種センサキャリブレーションの完了をタブの色で表示
- ブートデバイス内の設定ファイルを直接編集する機能を実装
- UADFの相対パスに対応
- RCコマンドのリモートインターフェースを追加
