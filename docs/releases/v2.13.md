# Release v2.13

## 変更点

---

### 制御

- 制御器の出力制約を撤廃
- 並進制御のゲインを目標推力によってスロットル
- 位置制御器のアンチワインドアップを実装
- 位置コマンドにフィードフォワード加速度を追加
- 飛行モード変更時に目標姿勢を滑らかに遷移

### ミッション

- ミッションの高度は起動地点ではなくアーム地点を基準とする
- ミッションの上書き・キャンセル時は一度滑らかに停止

### マニュアルモード

- ディスアーム指令でモータを即停止させる
- キルスイッチを入れたままマニュアルモードを有効にできないように
- プロポからのコマンドの変化率に制約をかける

### その他

- GPSマークを変更
- GCSのメッセージビューアを改善
- ネットワークインターフェースを固定
- ROSのログをストレージではなくRAM上に出す
