# Release v2.12

## Changes

---

### Mission

- Default values can now be specified for mission parameters
- Default parameters can now be overridden on the mission planning screen
- RTL now executes through landing
- If a new mission arrives during mission execution, it interrupts according to priority
- Missions can no longer be executed when manual mode is enabled

### Ground Control Station

- Changed the map from OSM to aerial imagery
- Implemented map overzoom functionality
