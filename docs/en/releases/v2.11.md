# Release v2.11

## Changes

---

### State Estimation

- Takeoff is determined using only ground reaction force, since acceleration or velocity can easily cause false detection

### Control

- Integral error is reset on disarm rather than when landing is detected
- Implemented rate mode for Y-axis tilt multi

### Mission

- Unified the action servers for mission execution into a single node
- Send all mission commands to the FC before execution
- Run Mission Executor as a Component
- In the Land command, inherit the previous target latitude and longitude

### Manual Mode

- Use `vehicle_health` as the criterion for determining whether a mode change is allowed

### Flight Log

- Implemented CSV export for flight logs

### Other

- Implemented the action interface in `tobas_ros_interface`
- Added a remote IF for `external_pose`
