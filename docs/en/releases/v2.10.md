# Release v2.10

## Changes

---

### State Estimation

- Sensors used by the ESKF can now be selected in Setup Assistant
- External Pose can now be input to the ESKF
- Changed the conditions for takeoff and landing detection

### Failsafe

- The threshold for ESC communication loss detection can now be changed
- After target thrust starts being output, disarm on a 500 ms communication loss without waiting 10 s

### Communication

- The network interface can now be specified in `CYCLONEDDS_URI`

### Flight Log

- Added an RC Input tab

### Simulation

- Added a suspended payload plugin

### Other

- Time-consuming processing such as sending and building is now done in a separate thread
- If the build error message is long, it is output to a file
- Floating-point output in yaml is rounded appropriately
