# Release v2.14

## Changes

---

### Sensor Processing

- Enabled BDU (Block Data Update) for the IMU, magnetometer, and barometer

### Control

- Changed the operating frequency of the entire system from 400 Hz to 800 Hz
- Changed the default damping ratio for horizontal position control from 1.0 to 0.7
- Added support for specifying the center pitch angle for engine-powered aircraft (default: 0 deg)

### Failsafe

- Added support for user-defined check items in the Pre-Arm Check

### Flight Log

- Added PWM throttle
- Removed the 10 GB capacity limit

### Simulation

- Added a simulation process monitoring feature

### Other

- Clear the cache when building a local project
- Publish smoothed CPU temperature and load
