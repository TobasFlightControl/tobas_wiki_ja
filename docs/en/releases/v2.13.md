# Release v2.13

## Changes

---

### Control

- Removed controller output constraints
- Throttle translational control gains based on target thrust
- Implemented anti-windup for the position controller
- Added feedforward acceleration to position commands
- Smoothly transition target attitude when changing flight modes

### Mission

- Mission altitude is now referenced to the arm position instead of the startup position
- Smoothly decelerate once when overwriting or canceling a mission

### Manual Mode

- Stop the motors immediately on a disarm command
- Prevent enabling manual mode while the kill switch is on
- Limit the rate of change of commands from the RC transmitter

### Other

- Changed the GPS marker
- Improved the GCS message viewer
- Fixed the network interface
- Output ROS logs to RAM instead of storage
