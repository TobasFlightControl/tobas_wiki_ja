# Release v2.7

## Changes

---

### Sensor Processing

- Changed the compass calibration method from max-min to ellipsoid fitting

### Modeling

- Implemented estimation of engine dynamics
- Implemented estimation of aerodynamic characteristics for variable-pitch propellers

### Control

- Added support for multicopters that tilt only around the Y-axis
- Improved the mixer for multicopters with non-planar rotor arrangements
- Multicopter gains can now be configured not only with the standard second-order form, but also with a two-stage cascade

### Communication

- Changed the connection from GCS to FC from login/password to SSH key authentication
- Fixed a bug where ROS communication would fail if the simulation was restarted immediately after ending

### Other

- Display completion of various sensor calibrations using tab colors
- Implemented a feature to directly edit configuration files on the boot device
- Added support for relative paths in UADF
- Added a remote interface for RC commands
