# Release v2.9

## Changes

---

### Control

- Automatically disarm upon landing detection

### Mission

- Improved path planning for waypoint flight

### Failsafe

- Added alignment with the geomagnetic reference vector to the checklist

### Manual Mode

- Assume the throttle lever is reversed to match Futaba
- Changed the RC switch channel assignment (CH5 for flight mode)
- Reduce RPM at zero throttle

### Ground Station

- Improved layout
- Display GNSS status

### Flight Log

- Display ESC communication errors
- Display CPU status

### Simulation

- Added a Gazebo world
- Enabled FTDI USB device selection in the GUI

### Other

- Reduced messages output under `/rosout` and `~/.ros/log/`
- Calculate and publish vibration levels from accelerometer values
- Changed the world coordinate system from NWU to ENU
- Added Wi-Fi key management types
- Display the version in the GUI title
- Tobas project version check
