# Release v2.8

## Changes

---

### Sensor Processing

- Implemented compass calibration using WMM for large aircraft

### Communication

- Implemented a communication check feature from FC to GCS

### Ground Control Station

- Stabilized screen updates in Qt event processing
- Show a dialog when the GCS is disconnected from the network
- Changed the behavior of the GCS power button. Keep the GCS running and shut down only the FC.

### Flight Logs

- Added user joint status

### Simulation

- Reflected propeller inertial forces and gyroscopic effects
- Changed the Gazebo joint controller from `ros2_controllers` to a custom plugin

### Other

- Changed the specifications of the user Python package
- Implemented an option not to use the built-in controller
- Build the project in a separate thread
- Changed the ROS 2 installation layout on the Raspberry Pi side from `isolated` to `merged`
